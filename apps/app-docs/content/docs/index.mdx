## Welcome to RAG Chat

RAG Chat is an AI-powered text summarization and chat application that allows users to process documents, create notes, and interact with their content through an intelligent chatbot interface.

## What is RAG Chat?

RAG (Retrieval-Augmented Generation) Chat combines the power of:

- **Document Processing**: Upload and process text documents, PDFs, and other content
- **Text Chunking**: Intelligently split documents into manageable chunks for processing
- **Vector Embeddings**: Generate embeddings for semantic search and retrieval
- **AI Chat Interface**: Chat with your documents using advanced language models
- **Note Management**: Organize and manage your processed content

## Key Features

### AI Text Summarizer

Process and summarize long-form text content with AI-powered analysis. The application handles:

- Text input and document upload
- Automatic chunking and processing
- Vector embedding generation
- Status tracking for processing jobs

### Intelligent Chat

Chat with your notes using a sophisticated RAG pipeline:

- Context-aware responses based on your documents
- Vector similarity search for relevant information
- Streaming responses for real-time interaction
- Chat history and conversation management

### User Authentication

Secure authentication powered by Neon Auth:

- Email-based authentication
- Protected routes and API endpoints
- User session management

## Tech Stack

- **Frontend**: Next.js 16 with React 19
- **Authentication**: Neon Auth
- **Database**: PostgreSQL with Prisma ORM
- **AI/ML**:
  - OpenAI for embeddings and chat
  - LangChain for RAG pipeline
  - Anthropic Claude for advanced reasoning
- **Deployment**: Vercel-ready configuration
